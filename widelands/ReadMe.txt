Quick Guide.

pip3 install python-xlib pillow mss

Chuck the widelands directory into the autokey module directory.

Edit current_tribe.py with a number 0 to 4 (more if there are more tribes available) this can NOT detect if your playing as an Amazon or an Empire so this is critical for functioning.

Edit NOTIFICATIONS_DIR with your own sounds or pointing to where you saved the Notification directory.

WORK_PATH = '/dev/shm/Widelands/' to whatever you want, this is constantly being written to when creating roads etc. so /dev/shm is the best place if you have ssd.

see https://github.com/lazy-1/widelands-autokey-macros/tree/main/Tutorials for more comprensive guides.
